# Cleaning Service API

A RESTful API for managing cleaning service bookings and checking cleaner availability. Built with Spring Boot 3.5.8 and Java 21. Uses h2 embedded relational database. OpenAPI support with swagger UI is included. 


## Prerequisites

- Java 21


## Building the project 

```bash
./gradlew build
```

## Running the Application

```bash
./gradlew bootRun
```

The application will start on `http://localhost:8080`

## Access API Documentation

Once the application is running, you can access:
http://localhost:8080/swagger-ui.html
http://localhost:8080/api-docs
